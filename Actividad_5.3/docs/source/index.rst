Documentacion del Proyecto - Actividad 5.3
==========================================

Este proyecto compara dos estrategias de procesamiento de datos de posicion de la ISS
mediante una interfaz grafica en tiempo real construida con Tkinter.

Objetivos principales
---------------------

* Comparar una version no optimizada frente a una optimizada.
* Mostrar metricas de tiempo de ejecucion y profiling.
* Visualizar la evolucion temporal de rendimiento y latitud con graficos.
* Mantener fluidez de la interfaz usando tareas en segundo plano.
